# Infinity Expansion

InfinityExpansion is an addon to the plugin Slimefun which adds a bunch of new machines and resources.
It expands the endgame of Slimefun by a large margin.

## Requirements

- Java 21+
- Paper 1.21+
- [Slimefun5](https://github.com/Slimefun/Slimefun5)
