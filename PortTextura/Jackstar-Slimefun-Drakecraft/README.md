# Jackstar — Slimefun (DrakesCraft Labs)

Resource pack mantenido en el monorepo **drakes-slimefun-labs** (`PortTextura/Jackstar-Slimefun-Drakecraft`) para servidores **Paper / Purpur 1.21.x** con **Slimefun** (línea Drake) y addons habituales.

## Qué incluye

- Modelos y texturas Slimefun (y muchos addons), GUI, CIT OptiFine / Cit Resewn donde aplica.
- Namespace de modelos extra: **`jackstar`** (antes `racingsky` en el pack original de la comunidad).
- **`pack_format: 34`** para clientes **Java 1.21–1.21.1**. La carpeta `assets/minecraft/items/` es para definiciones de ítem de **1.21.4+**; en 1.21.1 el juego usa sobre todo `assets/minecraft/models/item/` + `custom_model_data`.

## `item-models.yml`

Es la tabla de **CustomModelData** para el core Slimefun. Tras el primer arranque del plugin, suele generarse `plugins/Slimefun/item-models.yml`: puedes **fusionar** o **sustituir** con cuidado según tu servidor (haz copia antes).

## Créditos

- **DrakesCraft Labs** — fork y branding **Jackstar** para nuestra infraestructura.
- Base visual ampliamente derivada del trabajo comunitario conocido como **Racingsky** / packs Slimefun circulantes; se conserva atribución en metadatos `credit` / `author` donde correspondía.
